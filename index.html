Pengantar Web Mining

Web Mining adalah proses penemuan pola dari data yang besar dan kompleks yang terdapat di web. Dalam mata kuliah ini akan dipelajari dasar-dasar crawling, preprocessing, text mining, hingga penerapan metode analisis.

Tujuannya agar mahasiswa memahami cara mengolah informasi dari web untuk keperluan riset maupun aplikasi nyata.
