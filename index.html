Pengantar Web Mining

Materi ringkas untuk halaman tugas:

  • Definisi Web Mining & ruang lingkup: Content, Usage, dan Structure Mining.
  • Tantangan: skala data, dinamika web, dan heterogenitas format (HTML, XML, teks).
  • Alur umum: Pengumpulan data → Preprocessing → Pembobotan/Representasi → Pemodelan → Evaluasi.
  • Contoh teknik: Naive Bayes & SVM (klasifikasi), K-Means (clustering), PageRank (structure), analisis clickstream (usage).
  • Contoh aplikasi: rekomendasi produk, analisis sentimen ulasan, deteksi komunitas, personalisasi pencarian.

Silakan sesuaikan isi sesuai instruksi dosen dan referensi materi kuliah.
